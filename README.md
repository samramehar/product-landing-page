#  Product Landing Page – FreeCodeCamp Responsive Web Design Project

This is my **Product Landing Page project** built as part of the FreeCodeCamp Responsive Web Design Certification. The project demonstrates my ability to create a responsive, modern landing page using only HTML and CSS.

---

##  Project Overview

This project is a sample **product landing page** that simulates a real-world marketing website. It includes sections like:

- Hero section with product introduction
- Features section
- How it works section
- Pricing or call-to-action section
- Footer with links

The goal was to build a responsive, user-friendly landing page following FreeCodeCamp project requirements.

---

##  Built With

- HTML5
- CSS3
- Flexbox
- Media Queries (Responsive Design)

---

##  Features

- Fully responsive layout (mobile + desktop)
- Clean and modern UI design
- Structured sections for product presentation
- Call-to-action button
- Smooth layout using Flexbox
- Simple and user-friendly interface

---

##  What I Learned

Through this project, I improved my skills in:

- Structuring landing page layouts
- Using semantic HTML elements
- CSS Flexbox for alignment
- Responsive design with media queries
- Building real-world UI components
